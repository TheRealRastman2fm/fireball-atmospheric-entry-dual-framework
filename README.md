# Fireball Atmospheric Entry Physics (RBEF Project)

This repository contains MATLAB scripts and NASA CNEOS fireball dataset analysis for meteoroid atmospheric entry, drag physics, and energy dissipation.

## How to run
Open MATLAB and execute:
main_simulation.m

## Outputs
- Velocity distribution
- Altitude statistics
- Energy correlation
- Trajectory simulation
